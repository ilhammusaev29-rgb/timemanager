# TimeManager

Персональная система тайм-менеджмента: веб-приложение + FastAPI бэкенд + Telegram-бот.

## Структура проекта

```
timemanager/
├── index.html      — фронтенд (HTML/CSS/JS, один файл)
├── api.py          — FastAPI: CRUD задач, занятий, статистика, экспорт
├── bot.py          — Telegram-бот (aiogram 3.x)
├── scheduler.py    — APScheduler: напоминания о занятиях
├── database.py     — SQLAlchemy модели и операции с БД
├── main.py         — точка входа (API + бот в одном процессе)
├── requirements.txt
├── .env.example    — шаблон переменных окружения
└── README.md
```

## Быстрый старт

### 1. Клонирование и установка зависимостей

```bash
git clone <repo>
cd timemanager
python -m venv venv
# Windows:
venv\Scripts\activate
# Linux/Mac:
source venv/bin/activate

pip install -r requirements.txt
```

### 2. Настройка окружения

```bash
cp .env.example .env
# Откройте .env и заполните BOT_TOKEN (получить у @BotFather в Telegram)
```

### 3. Запуск

```bash
# Запуск бэкенда и бота (в одном процессе):
python main.py

# ИЛИ — только API (без бота):
uvicorn api:app --reload --host 0.0.0.0 --port 8000

# ИЛИ — только бот:
python -c "import asyncio; from bot import create_bot_and_dispatcher; bot, dp = create_bot_and_dispatcher(); asyncio.run(dp.start_polling(bot))"
```

### 4. Открытие фронтенда

Откройте `index.html` в браузере. При доступном API (localhost:8000) приложение автоматически переключается в режим работы с сервером (зелёная точка в шапке). Иначе данные хранятся в localStorage.

## API документация

После запуска доступна по адресу: http://localhost:8000/docs

### Эндпоинты

| Метод  | Путь                | Описание                     |
|--------|---------------------|------------------------------|
| GET    | /tasks              | Список задач (фильтры)       |
| POST   | /tasks              | Создать задачу               |
| GET    | /tasks/{id}         | Получить задачу              |
| PATCH  | /tasks/{id}         | Обновить задачу              |
| DELETE | /tasks/{id}         | Удалить задачу               |
| GET    | /lessons            | Список занятий               |
| POST   | /lessons            | Создать занятие              |
| PATCH  | /lessons/{id}       | Обновить занятие             |
| DELETE | /lessons/{id}       | Удалить занятие              |
| GET    | /stats              | Статистика                   |
| GET    | /export             | Экспорт данных в JSON        |

### Параметры задачи

- `title` — название
- `priority` — `high` / `medium` / `low`
- `deadline` — ISO datetime (опционально)
- `status` — `planned` / `in_progress` / `done`
- `is_urgent` — bool (для матрицы Эйзенхауэра)
- `is_important` — bool
- `scheduled_time` — `"HH:MM"` (для плана дня)

### Параметры занятия

- `title` — название
- `lesson_date` — дата `YYYY-MM-DD`
- `lesson_time` — время `HH:MM`
- `recurrence` — `once` / `weekly`
- `remind_minutes` — за сколько минут напомнить (по умолчанию 15)

## Telegram-бот

### Привязка аккаунта

Напишите боту `/start` — ваш `chat_id` сохранится в БД, и напоминания начнут работать.

### Команды

| Команда  | Описание                                |
|----------|-----------------------------------------|
| `/start` | Привязка аккаунта                       |
| `/today` | Расписание занятий и задач на сегодня   |
| `/list`  | Занятия на ближайшие 7 дней             |
| `/add`   | Добавить занятие (пошаговый диалог)     |

### Напоминания

Планировщик проверяет занятия каждую минуту. Уведомление отправляется за `remind_minutes` до начала занятия по часовому поясу `Asia/Almaty`.

## Переменные окружения

| Переменная     | Описание                           | По умолчанию                    |
|----------------|------------------------------------|---------------------------------|
| `BOT_TOKEN`    | Токен Telegram-бота                | обязательно                     |
| `DATABASE_URL` | URL базы данных                    | `sqlite:///./timemanager.db`    |
| `API_HOST`     | Хост FastAPI                       | `0.0.0.0`                       |
| `API_PORT`     | Порт FastAPI                       | `8000`                          |

## Фронтенд: функциональность

- **Задачи** — добавление, редактирование, удаление; фильтрация по статусу/приоритету, поиск, сортировка
- **Матрица Эйзенхауэра** — 4 квадранта (срочно/важно)
- **Pomodoro** — настраиваемые интервалы, звуковой сигнал, кольцо прогресса
- **План дня** — задачи по часам, перетаскивание drag&drop
- **Статистика** — выполнено за день/неделю, % завершения, прогресс-бар
- **Темы** — светлая / тёмная с переключателем
- **Экспорт** — выгрузка всех данных в JSON
- **Адаптивность** — работает на мобильных устройствах
